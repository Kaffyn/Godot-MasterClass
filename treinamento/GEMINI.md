# Gemini CLI - Guia Central de Desenvolvimento de Jogos

Este é o documento central para o treinamento do Gemini CLI em desenvolvimento de jogos. Ele serve como um índice para os guias especializados que cobrem os tópicos em profundidade.

## 1. Planejamento de Jogos e Sistemas

A fundação de qualquer projeto de sucesso. Este guia ensina como transformar uma ideia em um plano estruturado, desde o conceito do jogo até a arquitetura de sistemas modulares.

- **Tópicos:** Game Design Document (GDD), Vertical Slice, Roadmaps, e o planejamento prático de um novo sistema (como um Sistema de Reputação) usando a arquitetura Server-Resource-Node do Zyris.

➡️ **[Acesse o guia detalhado: `game_planning.md`](./game_planning.md)**

## 2. Teoria e Design de Jogos

A base para criar experiências interativas memoráveis. Este guia cobre os "porquês" por trás de um bom jogo.

- **Tópicos:** Teoria das Cores, Psicologia das Cores, Game Design (Core Loops, Curva de Dificuldade), e Level Design (Fluxo, Pacing, Landmarking).

➡️ **[Acesse o guia detalhado: `game_design.md`](./game_design.md)**

## 3. Paradigmas de Programação para Jogos

A forma como estruturamos nosso código tem um impacto massivo na manutenibilidade e na performance do jogo.

- **Tópicos:** Programação Orientada a Objetos (OOP) com seus 4 pilares (Encapsulamento, Abstração, Herança, Polimorfismo) e Design Orientado a Dados (DOD) para alta performance.

➡️ **[Acesse o guia detalhado: `programming_paradigms.md`](./programming_paradigms.md)**

## 4. Arquitetura de Framework (Princípios Zyris)

Uma análise aprofundada da arquitetura Server-Resource-Node que forma a espinha dorsal do desenvolvimento de alta performance e escalável.

- **Tópicos:** A responsabilidade de Servers como verdade global, Resources como DNA do gameplay, e Nodes como orquestradores de cena.

➡️ **[Acesse o guia detalhado: `zyris_architecture_principles.md`](./zyris_architecture_principles.md)**

## 5. IA e Machine Learning para Jogos

Técnicas avançadas para criar inimigos e sistemas dinâmicos, com um foco no pipeline de treinamento e execução.

- **Tópicos:** Algoritmos de IA clássicos (A\* Pathfinding, Behavior Trees), técnicas de Machine Learning (Reinforcement Learning) e o fluxo de trabalho para treinar modelos em Python e usá-los em uma engine de jogo.

➡️ **[Acesse o guia detalhado: `zyris_ai_development.md`](./zyris_ai_development.md)**

## 6. Desenvolvimento com Godot Engine

Um guia prático para usar a Godot Engine, alinhado com os princípios de arquitetura do Zyris.

- **Tópicos:** A estrutura de Cenas e Nós, GDScript, o sistema de `Resources`, e o uso de C++ com GDExtension.

➡️ **[Acesse o guia detalhado: `godot_development.md`](./godot_development.md)**
