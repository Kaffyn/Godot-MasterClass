# Módulo 09: Shaders & VFX (Juice)

> **Foco:** Tech Art, Polimento e "Game Feel".

Um jogo funcional não é o bastante. Ele precisa ser bonito e "suculento".

## Ementa

1. **Shaders (GDShader):**

   - Vertex vs Fragment Shaders.
   - Criando efeitos visuais (Água, Fogo, Distorção).
   - Visual Shader Editor.

2. **Sistemas de Partículas:**

   - `GPUParticles2D` e `3D`.
   - Process Materials e Texturas.

3. **Pós-Processamento:**

   - `WorldEnvironment`.
   - Glow (Bloom), Fog, Color Correction (LUTs).

4. **Otimização Gráfica:**
   - Draw Calls e Batching.
   - Occlusion Culling.
   - LOD (Level of Detail).
