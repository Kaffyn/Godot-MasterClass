# Módulo 05: Procedural Generation & Tilemaps (Roguelike)

> **Foco:** Algoritmos, Ferramentas de Editor e Rejogabilidade.

Como criar jogos infinitos? Vamos mergulhar na geração procedural e na criação de ferramentas para o editor.

## Ementa

1. **TileMaps Avançados:**

   - TileMapLayers.
   - Terrains (Autotile) e Bitmasking.

2. **Algoritmos de Geração:**

   - Random Walk (Cavernas simples).
   - Cellular Automata (Cavernas orgânicas).
   - BSP (Salas e Corredores).

3. **Pathfinding:**

   - NavigationServer 2D.
   - A\* (A-Star) para inimigos em labirintos dinâmicos.

4. **Tooling (`@tool`):**
   - Criando scripts que rodam no editor para facilitar o Level Design.
