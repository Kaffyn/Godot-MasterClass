# UI Responsiva: Containers e Ancoragem

Criar UI que funciona em qualquer resolução (celular, 4K, ultrawide) é uma arte.
Na Godot, a regra é: **Nunca posicione coisas na mão.** Use Containers.

---

## 1. Control Nodes e Ancoragem

Todo nó `Control` (o verdinho) vive dentro de um retângulo (Rect).
As **Anchors** (Âncoras) definem a qual parte do pai o nó está preso.

- **Top-Left:** Padrão. Fica no canto.
- **Center:** Fica no meio.
- **Full Rect:** Ocupa todo o espaço do pai (ótimo para fundos).

Se você quer que uma barra de vida fique sempre no topo da tela, ancore-a no **Top-Wide**. Se a tela esticar, a barra estica junto.

---

## 2. O Poder dos Containers

Containers são nós que controlam a posição e o tamanho dos seus filhos.
Se você colocar um botão dentro de um Container, você **não pode** mais mover o botão manualmente. O Container é o chefe.

### Os Essenciais

- **HBoxContainer:** Organiza filhos em linha horizontal.
- **VBoxContainer:** Organiza filhos em coluna vertical.
- **GridContainer:** Organiza em grade (linhas e colunas).
- **MarginContainer:** Adiciona uma borda (padding) ao redor do filho.
- **CenterContainer:** Centraliza o filho perfeitamente.

### Layout Complexo? Combine-os

Quer um menu centralizado com título e botões?

1. `CenterContainer` (Preenche a tela toda).
2. Dentro dele, um `VBoxContainer`.
3. Dentro do VBox: `Label` (Título) e 3 `Buttons`.

Pronto. Responsivo para sempre.

---

## 3. Size Flags (O comportamento do filho)

Dentro de um Container, cada filho tem configurações de **Size Flags** (no Inspector > Layout > Container Sizing).

- **Fill:** "Ocupe todo o espaço disponível na direção oposta aos irmãos".
- **Expand:** "Empurre os irmãos para ganhar mais espaço".
- **Shrink Center:** "Fique no meio do meu espaço alocado".

Dominar Size Flags é a chave para UIs profissionais.

---

## 4. Temas (Themes)

Não estilize cada botão individualmente. Crie um **Theme Resource**.
O Tema define a fonte, as cores e as bordas de **todos** os botões do jogo.
Se você mudar a cor no Tema, muda no jogo inteiro.

> **Machi Way:** UI é separada da Lógica. A Barra de Vida não calcula a vida; ela apenas **mostra** o valor que recebe. Mantenha a UI "burra".
