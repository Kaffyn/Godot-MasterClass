# Módulo 08: Networking & Multiplayer (Arena)

> **Foco:** Conectividade, Sincronização e Autoridade.

O desafio final da engenharia: fazer o jogo rodar igual em duas máquinas diferentes.

## Ementa

1. **Fundamentos de Rede:**

   - TCP vs UDP.
   - High-level Multiplayer API da Godot.

2. **Arquitetura Cliente-Servidor:**

   - Servidor Dedicado vs Listen Server.
   - Autoridade (Quem decide o que é verdade?).

3. **Sincronização:**

   - `MultiplayerSynchronizer`.
   - RPCs (Remote Procedure Calls).
   - Spawnando jogadores na rede.

4. **Polimento de Rede:**
   - Interpolação de movimento.
   - Lidando com latência (Lag Compensation básico).
