# Módulo 06: 3D Fundamentals (FPS Retro)

> **Foco:** A transição para o eixo Z e o mundo 3D.

Bem-vindo à terceira dimensão. Vamos criar um FPS estilo Doom/Quake para aprender os fundamentos do 3D na Godot.

## Ementa

1. **O Mundo 3D:**

   - `Node3D`, Transform3D (Posição, Rotação, Escala).
   - Eixos X, Y, Z (Regra da Mão Direita).

2. **Character Controller 3D:**

   - `CharacterBody3D`.
   - Câmera em Primeira Pessoa (Mouse Look).
   - Movimento e Gravidade.

3. **Level Design 3D:**

   - GridMaps (Tilemaps 3D).
   - CSG (Constructive Solid Geometry) para prototipagem rápida.
   - Importação de modelos `.glb` (Blender).

4. **Interação 3D:**
   - Raycasting 3D (Atirando).
   - Física 3D básica.
