# Módulo 08: Jogo de Ação Multiplayer (Overwatch 2D-like)

> **Foco:** Criar um jogo competitivo online com heróis únicos.

## Ementa

1. **Base de Herói:** Classes e herança para personagens únicos.
2. **Definição por Resource:** Armas e Habilidades configuráveis.
3. **Multiplayer Local:** Split-screen e controle de input múltiplo.
4. **Servidor Node.js (Híbrido):** Orquestrando partidas e matchmaking com TypeScript.
5. **Netcode Prático:** Sincronizando movimento e ações em tempo real.
