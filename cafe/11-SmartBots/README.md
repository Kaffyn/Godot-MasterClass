# Módulo 11: Bots Inteligentes

> **Foco:** Aplicando a IA treinada em um jogo real.

## Ementa

1. **Ambiente de Treinamento:** Configurando Godot como ambiente de simulação.
2. **Criando o Cérebro:** Implementando a lógica de aprendizado em Python.
3. **Treinamento:** Rodando sessões de treinamento aceleradas.
4. **Deploy da IA:** Exportando o modelo para GDExtension.
5. **Leaderboards:** Implementando ranking global conectado ao backend.
