# Módulo 02: Laboratório Top-Down

> **Foco:** Construir um jogo completo aplicando os fundamentos.

## Ementa

1. **Player Controller:** Movimento em 8 direções e normalização de vetores.
2. **Sistema de Câmera:** `Camera2D`, Smoothing e Limits.
3. **Inimigos Simples:** State Machine finita (Patrulha/Perseguição).
4. **Combate Básico:** Hitboxes, Hurtboxes e Projéteis.
5. **Coletáveis:** Moedas, Vida e Inventário simples.
6. **Game Manager:** Gerenciando Score e Game Over (Singleton).
