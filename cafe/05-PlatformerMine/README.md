# Módulo 05: Plataforma e Mineração (Dome Keeper-like)

> **Foco:** Jogo de plataforma complexo com sistemas de RPG.

## Ementa

1. **Platformer Controller Pro:** Coyote Time, Jump Buffer, Aceleração/Fricção.
2. **Mineração:** TileMaps destrutíveis e persistência de alterações.
3. **Ondas de Inimigos:** Sistemas de spawn controlados por tempo/eventos.
4. **Coleta e Crafting:** Loop de recursos usando os Resources do Módulo 04.
5. **Defesa da Base:** Construção de torretas e barreiras.
