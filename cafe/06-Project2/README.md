# Módulo 06: Projeto Autoral 2 (Desafio Cumulativo)

> **Foco:** Combinar mecânicas de plataforma com sistemas de dados complexos.

## Opções

### A: Plataforma de Precisão (Celeste-like)

- **Foco:** Game Feel perfeito.
- **Desafio:** Mecânicas avançadas (Dash, Wall Jump, Climbing).

### B: Metroidvania Compacto

- **Foco:** Mapa interconectado e Backtracking.
- **Desafio:** Habilidades que desbloqueiam novas áreas.

### C: Simulador de Fazenda Top-Down

- **Foco:** Sistemas de gerenciamento (Plantio, Colheita, Dia/Noite).
- **Desafio:** Salvar o estado complexo da fazenda.

### D: Roguelike Dungeon Crawler

- **Foco:** Geração Procedural.
- **Desafio:** Mapas aleatórios e morte permanente.
