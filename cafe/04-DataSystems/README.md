# Módulo 04: Sistemas de Dados e Progressão

> **Foco:** Resource-Oriented Programming (ROP) e arquitetura de dados escalável.

## Ementa

1. **Resource Architecture:** Criando dados desacoplados (`ItemData`, `SkillData`).
2. **Sistema de Inventário:** Dados vs UI (MVC).
3. **Diálogos e Missões:** Estruturas de dados para narrativa não-linear.
4. **Lojas e Economia:** Comércio e transações.
5. **Progressão (RPG):** Curvas de XP e Level Up.
6. **Animação Avançada:** `AnimationTree` e State Machines de animação.
